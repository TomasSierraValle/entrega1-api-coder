# Entrega 2 — WebSockets + Handlebars (Coderhouse)

## Descripción
Proyecto de práctica del curso **Backend con Node.js (Coderhouse)**.  
Se implementa un servidor con **Express**, **Handlebars** como motor de vistas y **Socket.io** para actualizar en tiempo real la lista de productos.

---

## Requisitos

- **Node.js 18+**
- **npm** o **yarn**
- Git instalado
- Handlebars
